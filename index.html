Mission

An academic lab site should be readable in bright mode and memorable in dark mode.

Use dark mode for first impression, keynote screens, project showcases, and lab branding. Use bright mode for long reading, faculty review, committee browsing, and screenshots inside application packets. Because both modes are implemented with the same design system, you do not need two different websites.
